# fo2-fake-server

### fPrerequisite
**Make sure you have latest NodeJs installed on**

https://nodejs.org/en/download/

**Steps to setup Fo2 Fake Server**
```
Step-1 : git clone https://github.com/githuboxy/fo2-fake-server.git 
Step-2 : cd fo2-fake-server
Step-3 : npm install
Step-4 : npm start
```
